# 朴素集合论

一个数学理论是一个被表达的思维模式.
集合论比较接近语言学, 基于 ZFC 公理化集合论的数学, 是数学领域目前的主流

集合即一类事物的代名词.
对于一个集合, 任何一个被讨论的事物, 都属于或不属于它

本书介绍的朴素集合论, 介绍集合论的通用概念, 以便理解 ZFC 等更深入的话题

---

## 第一章  集合

第一节的内容, 和高中教材对集合的表述, 基本重合
它的主要的目的是, 介绍集合的基本概念, 便于更多的读者能轻松地阅读此书

稍后会讨论一些定义和证题, 怎么把有穷集领域的概念表述, 推广到无穷集
