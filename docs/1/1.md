# 1. 集合

## 数集

数字是数集的基本元素

以下是一些例子

$$\mathbb{N} = \{ 0, 1, 2, \cdots \}$$

$$\mathbb{Z} = \{ \cdots, -2, -1, 0, 1, 2, \cdots \}$$

$$\mathbb{Q} = \{ p/q, p \in \mathbb{Z}, q \in \mathbb{Z} \setminus \{ 0 \} \}$$

## 一些更强的子集关系

### 两集合相等

$A$ 和 $B$ 互为子集

$$A = B \Leftrightarrow A \subset B \wedge B \subset A$$

![1.1.5](../assets/images/1.1.5.JPG)

### 两集合为真子集关系

构成子集关系但不构成相等

$$A \subsetneqq B \Leftrightarrow A \subset B \wedge B \not\subset A$$

![1.1.6](../assets/images/1.1.6.JPG)

### 空集

空集是所有集合的子集

没有元素属于空集

## 从两个集合到一个新的集合的运算

### 交集

记作 $A \cap B$

![1.1.7](../assets/images/1.1.7.JPG)

### 并集

记作 $A \cup B$

![1.1.8](../assets/images/1.1.8.JPG)

### 差集

记作 $A \setminus B$

仅在其中一个集合的元素

![1.1.9](../assets/images/1.1.9.JPG)

### 对称差集

$A \bigtriangleup B$

严格属于一个集合的元素
