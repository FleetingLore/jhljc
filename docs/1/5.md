# 5. Cantor-Bernstein 定理

集合基数的不大于的关系, 有传递性, 至于等量关系沿着不大于关系的传递性, 是这个定理的必要引理

考虑 $A_2 \subset A_1 \subset A$, $A_2$ 与 $A_0$ 基数相等, 则对映射 $f: A_0 \to A _2$ 作递归, 得链

$$\cdots \subset A_3 \subset A_2 \subset A_1 \subset A_0$$

此处根据子集公理, 存在一个集合 $C$, 是所有链节的交集

通过差集运算得到族

$$\{ C_0, C_1, C_2, \cdots, C \}$$

奇数项一一对应, 偶数项后置对应, 得到 $A_0$ 和 $A_1$ 的一一对应
