## 子集关系的运算律

**自反性**

$$A \subset A$$

**传递性**

$$A \subset B \wedge B \subset C \implies A \subset C$$

![1.4.1](../assets/images/1.4.1.JPG)

**空集**

$$\emptyset \subset A$$

**等价关系的定义**

$$A \subset B \wedge B \subset A \iff A = B$$

第二章给出, 可以确认集合等价的另一种表述

## 仅一种运算的运算律

### 并集的运算律

**幂等律**

$$A \cup A = A$$

**交换律**

$$A \cup B = B \cup A$$

**结合律**

$$A \cup (B \cup C) = (A \cup B) \cup C$$

这意味着, 讨论若干集合的并集的时候, 不需要讨论他们如何整合

![1.4.2](../assets/images/1.4.2.JPG)

**与包含关系等价的特定情形**

$$A \subset B \iff A \cup B = B$$

![1.4.3](../assets/images/1.4.3.JPG)

### 交集的运算律

**幂等律**

$$A \cap A = A$$

**交换律**

$$A \cap B = B \cap A$$

**结合律**

$$A \cap (B \cap C) = (A \cap B) \cap C$$

讨论若干集合的交集的时候, 也不需要讨论他们如何整合

![1.4.4](../assets/images/1.4.4.JPG)

**与包含关系等价的特定情形**

$$A \subset B \iff A \cap B = A$$

![1.4.5](../assets/images/1.4.5.JPG)

单说运算律, 交集和并集是很像的

### 差集的运算律

**对自身的差集是空集**

$$A \setminus A = \varnothing$$

**与空集的差集**

$$A \setminus \varnothing = A$$

$$\varnothing \setminus A = \varnothing$$

**不服从交换律**

$$A \setminus B \neq B \setminus A$$

注意这个不等号的意思是并非对所有情形成立, 而不是对所有情形不成立.
此处用不等号, 即两边的表达式不能在一个表述中随意互换

**不服从结合律**

$$A \setminus (B \setminus C) \neq (A \setminus B) \setminus C$$

![1.4.6](../assets/images/1.4.6.JPG)

**与包含关系等价的特定情形**

$$A \subset B \iff A \setminus B = \varnothing$$

![1.4.7](../assets/images/1.4.7.JPG)

### 对称差的运算律

**交换律**

$$A \triangle B = B \triangle A$$

**结合律**

$$A \triangle (B \triangle C) = (A \triangle B) \triangle C$$

![1.4.8](../assets/images/1.4.8.JPG)

**空集是单位元**

$$A \triangle \varnothing = A$$

**自身是逆元**

$$A \triangle A = \varnothing$$

**与包含关系等价的特定情形**

$$A \subset B \implies A \bigtriangleup B = B \setminus A$$

![1.4.9](../assets/images/1.4.9.JPG)

## 含两种运算的运算律

### 交集和并集

**分配律**

$$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

$$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$

![1.4.10](../assets/images/1.4.10.JPG)

**吸收律**

$$A \cap (A \cup B) = A$$

$$A \cup (A \cap B) = A$$

![1.4.11](../assets/images/1.4.11.JPG)

### 差集与交集, 并集

**对左展开不变号**

$$(A \cup B) \setminus C = (A \setminus C) \cup (B \setminus C)$$

$$(A \cap B) \setminus C = (A \setminus C) \cap (B \setminus C)$$

![1.4.a](../assets/images/1.4.a.JPG)

**对右展开变号**

$$A \setminus (B \cup C) = (A \setminus B) \cap (A \setminus C)$$

$$A \setminus (B \cap C) = (A \setminus B) \cup (A \setminus C)$$

![1.4.13](../assets/images/1.4.13.JPG)

### 对称差集与交集, 并集, 差集

**交集分配**

$$A \cap (B \bigtriangleup C) = (A \cap B) \bigtriangleup (A \cap C)$$

![1.4.14](../assets/images/1.4.14.JPG)

**并集不分配**

$$A \cup (B \bigtriangleup C) \neq (A \cup B) \bigtriangleup (A \cup C)$$

![1.4.15](../assets/images/1.4.15.JPG)

**对称差的表示**

$$
\begin{aligned}
A \bigtriangleup B
& = (A \setminus B) \cup (B \setminus A) \\
& = (A \cup B) \setminus (A \cap B)  \\
\end{aligned}
$$

![1.4.16](../assets/images/1.4.16.JPG)

**差集的表示**

$$A \setminus B = A \bigtriangleup (A \cap B)$$

![1.4.17](../assets/images/1.4.17.JPG)

## 运算的完备性

表述两个集合的所有可能的运算结果, 三种就足够么?
会有覆盖不到的情形么?

作为运算结果, 讨论 Venn 图上独立的三个区域, 每个区可以选或不选, 那么结果的总数为 $$2 ^3 = 8$$

![1.4.18](../assets/images/1.4.18.JPG)

比如说用交集, 并集, 差集, 描述所有的运算结果, 表达式无非分成 $8$ 类, 而这四个运算, 连同他们表示的对称差集, 本身就不重复地占了 $4$ 类, 其中差集提供了两类, 集合的本身又是 $2$ 种情况
剩下的最后一个情况是空集, 可以来自任意集合对自身的自身差集

![1.4.19](../assets/images/1.4.19.JPG)

于是借助这三种运算, 可以表示二元集合运算的所有结果, 但能少一个么?
