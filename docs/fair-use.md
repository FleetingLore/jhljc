# 合理使用声明

本讲义（以下简称"本作品"）基于 **A. Shen, N. K. Vereshchagin** 所著、**陈光还** 翻译的集合论教材（以下简称"原书"）制作。

## 使用情况

| 内容 | 处理方式 |
|------|----------|
| 正文（讲解、证明、分析） | 完全独立重写 |
| 插图 | 全部重新绘制，且数量远超原书 |
| 习题 | 引用原书问题表述 |

## 合理使用依据

本作品对原书的使用属于《中华人民共和国著作权法》第二十四条规定的合理使用情形：

1. **转换性使用**：正文全部重新撰写，行文风格、组织结构、论证方式均与原书不同；插图全部独立重新绘制——本作品具有充分的独立创作性，并非原书的替代品。

2. **目的与性质**：本作品为教学讲义，属于为课堂教学和科学研究而对已发表作品的少量引用。

3. **使用比例**：仅习题部分保留原书表述，正文和插图均为原创。习题作为数学问题的标准陈述，其表达空间极为有限，属于思想的必要表达。

4. **市场影响**：本作品不会替代原书，反而引导读者阅读原书。

## 使用限制

- 本作品仅供学习参考，未经授权不得用于商业用途。
- 如需引用本作品内容，请注明出处。
